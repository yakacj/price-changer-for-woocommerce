Change all of your WooCommerce products prices very quickly, without edit product one by one.

== Description ==

This is a simple plugin for WordPress/WooCommerce store, to change all of your products prices by specific percentage when needed, 
without edit product price from admin product page.

Working for simple and variable products also as well.

This plugin does not change product prices from database ( it remain original prices, changes for frontend only), after changes, prices will show according to selected percentage and increase or decrease action. 
Prices will show and calculated all of WooCommerce pages, order and emails also, wherever product price applied/described.

== Installation ==

1. Upload "price-changer-for-woocommerce.zip" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. OR, Go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'price changer for woocommerce'.
4. Install and activate the plugin.
5. Plugin settings found beside activate/deactivate plugin on plugins page, also found on admin menu under woocommerce menu.
